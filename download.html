elementary OS Daily PPA (easier, but not recommended)

You can add the elementary OS Daily PPA to get the latest granite, but doing so will result in other unstable updates being installed, and is therefore not recommended.

$ sudo add-apt-repository ppa:elementary-os/daily
$ sudo apt-get update
$ sudo apt-get upgrade
Advanced Options
Daily (Unstable) PPA

Birdie can be installed from our Daily PPA, which provides automatic updates whenever we release improvements.

$ sudo add-apt-repository ppa:birdie-team/daily
$ sudo apt-get update
$ sudo apt-get install birdie
