Birdie for elementary

Birdie is a beautiful & fast Twitter client designed for elementary OS.

Download

  • Designed for elementary OS

    Birdie feels right at home in elementary OS with its beautiful interface and desktop integration.

  • Open Source

    Birdie is completely free and open source. Its source code is available for anyone to download and modify.

  • Endless Tweets

    You can browse your Twitter stream forever in Birdie with its infinitely scrolling interface.

  • 3rd Party Integration

    Birdie has built-in viewers for Instagram, YouTube, imgur and Vine.

Get Involved

Translations

Birdie is translated in dozens of languages by people from all around the world. Feel free to contribute at Transifex.

Code

Our code is available for everyone to check and contribute. If you think you might be able to help, feel free to send us a pull request at GitHub.

Bug Reports

If you have found a bug, we'd appreciate if you could report it over at GitHub. If you do so, you're helping us improve Birdie and fix bugs that we might not notice otherwise.
